Using with Firefox Containers

Firefox Multi-Account Containers let you separate your online identities. Combined with different proxies, you can have completely isolated browsing sessions.

1

Create Containers

From the 922Proxy extension popup, click "New Container" to create Firefox containers with random names and colors.

2

Switch Proxy Before Using Container

Before browsing in a container:

  1. Click the FoxyProxy icon and select a proxy
  2. Use the container buttons to "Open in Container" or "Switch To" existing tabs
  3. Each container can use a different proxy for complete isolation

Advanced Setup

For advanced users: You can configure FoxyProxy patterns to automatically use specific proxies for specific container tabs or websites.

Common Issues

  • Proxy not working: Check your 922proxy account status and data usage
  • Can't import config: Make sure you're using FoxyProxy Standard (not Basic)
  • Containers not showing: Install Multi-Account Containers extension
  • Extension not loading: Check that it's enabled in Firefox Add-ons
  • Credentials not saving: Make sure browser storage is enabled

Need Help?

If you're still having issues:

  • Check your 922proxy account dashboard for data usage and status
  • Try generating a new configuration with fewer proxies
  • Restart Firefox and try again
  • Contact 922proxy support for account-related issues
